How to read this

Scraped
Permit record found on the NLV EnerGov portal — number, type, status and dates captured. Service detail behind contact-login.
ArcGIS scraped
Permit existence confirmed via CLV / Henderson public ArcGIS REST permit layers (queried by APN). No amperage / voltage / phase / kVA field in the GIS schema — type + free-text description only. Does not substitute for an NV Energy confirmation.
🔒 detail gated
Permit found, but service size is locked. The amps / voltage / kVA tabs require a record-contact login. Two ways to unlock: (a) GPMG logs in as the permit contact, or (b) an NV Energy authorized request.
⬜ no elec schema
ArcGIS layer has no electrical fields. Only permit type and description are public; amps/voltage/kVA do not exist in this schema. NV Energy is the only path to these values.
Pending
Portal not yet run for that parcel (Clark Co. Accela requires OAuth registration).
⬜ kVA
Transformer rating is Unknown for every parcel until NV Energy confirms it — that is the only authoritative source, and closing it is the whole point of Stage 3.
